## 📌 Problem Statement  

Finding lost children in large cities is a challenging task due to the lack of immediate and efficient technologies. Currently, parents must file formal complaints and wait for assistance, which consumes valuable time.  

The **iSearch** application aims to address this issue by allowing parents to locate their lost children quickly using a mobile platform. Parents can submit images, provide detailed information, and search through a list of missing children. The system maintains a structured database that helps in retrieving relevant information efficiently.

---

## 🚀 Key Features  

- **User Module**  
  - Register/Login  
  - Submit missing person complaints  
  - Upload images & details  

- **Public Module**  
  - Register/Login  
  - View and search for missing persons  

---

## 📂 System Modules  

### 👤 User  
- **Login/Register**: Secure authentication for users  
- **Complaint Details**: Submit detailed reports of missing persons  
- **Upload Images**: Attach images to improve search efficiency  

### 🌍 Public  
- **Register/Login**: Secure access for general users  
- **View & Search**: Browse and search the database for missing individuals  

---

## 💡 How It Works  

1. Users register and log in to the system.  
2. Parents report missing children by providing details and uploading images.  
3. The database stores all missing person reports.  
4. The public can search for missing individuals through the app.
